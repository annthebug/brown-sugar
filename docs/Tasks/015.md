# Task 015 — 音訊系統（Howler.js）

狀態：🟨 進行中

## 目標
建立 SFX 音訊系統。

## 範圍
- 使用 Howler.js。
- SFX（風吹、貓叫、腳步、玻璃敲擊、火焰、Boss 等）。
- 音量（Sound）可於設定調整與記憶。
- **不包含背景音樂（BGM）。**

## 對應文件

## 實作步驟
1. 建立 `services/audio.ts`。
2. 建立音訊資源清單與預載。
3. 串接設定（音量）並存檔。
4. 處理行動裝置首次互動限制。
