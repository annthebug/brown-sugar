# Task 001 — 專案初始化（Vite + React + TypeScript）

狀態：⬜ 未開始

## 目標
建立基礎前端專案骨架，使用 Vite + React + TypeScript。

## 範圍
- 初始化 Vite（react-ts template）。
- 設定 ESLint / Prettier。
- 建立基礎目錄結構（見 11_TechnicalArchitecture）。

## 對應文件
- [11_TechnicalArchitecture](../11_TechnicalArchitecture.md)
- [13_Deployment](../13_Deployment.md)

## 實作步驟
1. `npm create vite@latest` 選擇 react-ts。
2. 安裝相依並確認 `npm run dev` 可啟動。
3. 建立 `src/` 內基礎資料夾：`components/`、`game/`、`stores/`、`services/`、`data/`、`routes/`。
4. 設定 lint 指令。

## 驗收標準
- `npm run dev` 可啟動並顯示首頁。
- `npm run lint` 通過。
