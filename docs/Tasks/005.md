# Task 005 — 主角黑糖：移動與能力

- 動畫：`bs-idle`、`bs-side-idle`、`bs-walk`、`bs-jump`、`bs-dash`、`bs-meow`、`bs-collect`、`bs-happy`、`bs-sad` 已註冊並串接狀態機。
