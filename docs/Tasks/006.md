# Task 006 — Memory 系統（收集與解鎖）

狀態：✅ 完成

## 目標
實作 Memory Shards 收集與回憶解鎖流程。

## 範圍
- 收集 Memory Shards（取代金幣）。
- 每 100 個解鎖一段回憶。
- 回憶動畫：Pixel → Fade → 照片 → 一句文字 → Continue → Pixel。

## 對應文件
- [06_GameMechanics](../06_GameMechanics.md)
- [03_Story](../03_Story.md)

## 實作步驟
1. 建立 Shard 物件與拾取邏輯。
2. 於 store 累計數量並觸發解鎖事件。
3. 以 GSAP 製作回憶播放動畫序列。

## 驗收標準
- 收集達 100 可正確解鎖回憶並播放動畫。
- 回憶存入相簿。

## 相依任務
- 003, 005

## 完成紀錄（2026-07-08）
- `useGameStore` 已支援 Memory Shards 累積、總收集數與每 100 shards 產生一次回憶解鎖事件。
- `GameScene` 已建立可點擊的 Memory Shard placeholder，透過 event bus 串接 React 收集流程；Task 005 完成後可替換為碰撞自動拾取。
- `useGalleryStore` 已支援解鎖下一段未解鎖回憶、保存解鎖狀態，Gallery 可顯示並重播已解鎖回憶。
- `MemoryOverlay` 已使用 GSAP 製作 Pixel → Fade → Photo → Text → Continue → Pixel 的柔和回憶播放流程。
