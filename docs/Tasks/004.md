# Task 004 — 資源預載與美術資源結構

狀態：✅ 完成

## 目標
建立資源目錄規範與 Phaser 資源預載流程。

## 範圍
- 建立 `assets/` 子結構（characters / scenes / ui / memories / bowls / audio）。
- 建立 `PreloadScene` 載入必要資源。
- 建立資源清單（manifest）。

## 對應文件
- [09_Assets](../09_Assets.md)

## 實作步驟
1. 依規範建立資源資料夾與命名。
2. 於 PreloadScene 載入 sprite atlas 與音訊。
3. 顯示載入進度畫面。
4. 建立淡藍天空與莫蘭迪色系 UI / scene 資源分類。

## 驗收標準
- 資源正確預載且有載入畫面。
- 資源命名符合規範。
- 載入畫面與預設場景資源使用淡藍天空背景與低飽和莫蘭迪配色。
