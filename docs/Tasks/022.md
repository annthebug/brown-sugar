- 新增 `vercel.json`：Vite preset、`npm run build`、`dist` 輸出、React Router SPA rewrite。
- 新增 `.env.example`：列出 Firebase `VITE_*` 變數，供本機與 Vercel 設定參考。

### Vercel 儀表板（需專案擁有者操作一次）
1. Import GitHub repo `annthebug/brown-sugar`。
2. Framework Preset 選 **Vite**（或由 `vercel.json` 自動偵測）。
3. 於 **Environment Variables** 設定 `.env.example` 中的 Firebase 變數（Preview / Production 皆可；未設定時遊戲仍可用 LocalStorage 遊玩）。
4. 推送 feature branch → 取得 Preview URL；merge 至 `main` → Production 部署。

### 驗證
- `npm run build` 通過。
- SPA 路由由 `vercel.json` rewrite 至 `index.html`，支援 `/game`、`/gallery` 等深層路徑重新整理。
