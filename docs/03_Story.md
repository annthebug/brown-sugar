# 03 — 故事（Story）

## 主線大綱

有一天，主人做了一個非常珍貴的玻璃抹茶碗。卻因為命運，玻璃碗化作許多 Memory Shards。黑糖決定展開旅程，一路收集碎片，找回所有回憶。

## 情感主軸

這不是一趟為了「物品」的旅程，而是為了「回憶」的旅程。玻璃碗只是引子，真正被拼回的是兩個人一路走過的故事。最終的體悟是：**有些禮物，不用完美，真正重要的是心意。**

## 章節敘事

### 序幕
玻璃抹茶碗誕生，卻碎裂成無數 Memory Shards，散落各地。黑糖踏上旅程。

### 第一章 — 森林（開始）
最初的探索。收集到第一段回憶：**第一次一起出去**（Memory #1）。基調：新鮮、雀躍。

### 第二章 — 城市（距離）
穿梭咖啡店、公園、捷運。面對 **Time Monster**，象徵遠距離與時間的考驗。收集 Memory #2。基調：思念。

### 第三章 — 雪山（旅行）
日本、雪景、合掌村。面對 **Snow Spirit**。收集 Memory #3。基調：一起走過遠方。

### 第四章 — 玻璃工坊（初次嘗試）
火爐、吹玻璃、玻璃管。擊敗 **Glass Master**，得到第一個玻璃碗——但不是想要的形狀。黑糖：🥺。基調：挫折。

### 第五章 — Retry（努力）
重新開始，重新收集材料。面對 **Inner Doubt**（內心的懷疑）。完成真正的玻璃碗。基調：堅持。

### 終章 — Final Stage（完美主義）
面對 **Perfectionism**。它不斷說：「不夠好。」「重做。」「再一次。」玩家擊敗它。
畫面浮現：**有些禮物，不用完美。真正重要的是：心意。**

## 結局

依 MBTI 顯示專屬玻璃碗（sprite sheet 裁切預覽）。黑糖躺在玻璃碗旁睡著，Credits 緩緩捲動，結束。

## 語氣與風格

- 溫暖、柔和、不說教。
- 台詞精簡、留白。
- 避免暴力與衝突，衝突以情緒隱喻呈現。
