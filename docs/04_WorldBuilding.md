# 04 — 世界觀（World Building）

## 世界基調

一個溫暖、柔和、Pixel 16bit 風格的世界。色調以暖色系為主，帶有 Pokémon Pixel Art 的親切感。世界不殘酷，衝突以情緒與隱喻呈現，而非暴力。

## 起源故事

有一天，主人做了一個非常珍貴的玻璃抹茶碗。卻因為命運，玻璃碗化作許多 Memory Shards，散落於各個世界角落。黑糖決定展開旅程，一路收集碎片，找回所有回憶。

## 世界地圖（章節區域）

| 區域 | 主題 | 象徵 |
| --- | --- | --- |
| Forest（森林） | 起點、自然 | 開始 |
| City（城市） | 咖啡店、公園、捷運 | 距離與時間 |
| Snow Mountain（雪山） | 日本、雪景、合掌村 | 旅行 |
| Glass Studio（玻璃工坊） | 火爐、吹玻璃、玻璃管 | 創造與挫折 |
| Retry（重製之地） | 重新收集材料 | 努力 |
| Final Stage | 內心之境 | 完美主義 |

## 世界規則

- **Memory Shards**：世界的核心資源，是回憶碎裂後的形態。收集足夠即可還原一段回憶。
- **回憶（Memory）**：以真實照片＋一句話呈現，是玻璃碗被還原的關鍵。
- **Boss**：非傳統敵人，而是情緒或阻礙的化身（時間、距離、懷疑、完美主義）。
- **玻璃碗**：世界的終極目標，最終依玩家人格（MBTI）呈現不同花紋。

## 氛圍設計

- 森林：風吹、鳥鳴、柔光。
- 城市：溫暖燈光、人聲環境音、捷運節奏。
- 雪山：靜謐、風雪、遼闊。
- 玻璃工坊：火焰、玻璃敲擊、暖橘色調。
- 內心之境：抽象、留白、以聲音與文字營造壓力後釋然。
