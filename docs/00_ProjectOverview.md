# 00 — 專案總覽（Project Overview）

## 專案名稱

**Quest for the Perfect Bowl**

## 一句話定位

一款專屬兩人的 Pixel RPG 網頁遊戲：操控貓咪「黑糖」收集回憶碎片，找回玻璃抹茶碗，並在過程中完成一次隱藏式 MBTI 分析。

## 核心主題

> 找回的不只是玻璃碗，而是一路走過的回憶。

## 專案目標

- 打造一份具有情感價值、專屬於兩人的互動禮物。
- 以遊戲化方式，讓玩家在無壓力、無察覺下完成 16 型人格分析。
- 將旅程中的「回憶」以照片＋一句話的形式收藏成相簿。
- 依 MBTI 結果，於結局生成獨一無二的玻璃碗。

## 玩家體驗流程

```
開始 → 森林 → 城市 → 雪山 → 玻璃工坊 → 重新製作 → 最後 Boss
     → 得到玻璃碗 → MBTI 結果 → Credits
```

## 最終產出（玩家獲得）

1. 專屬玻璃抹茶碗（依 MBTI 生成花紋）
2. MBTI Personality 結果
3. 一路收集的回憶相簿
4. Credits

## 遊戲類型標籤

- Pixel RPG
- Story Adventure
- Visual Novel
- Personality Test
- Memory Album

## 視覺方向

- 整體遊戲以淡藍色為主，背景基調為清透、柔和的淡藍色天空。
- 配色採莫蘭迪色系：低飽和灰藍作為主色，搭配霧粉、鼠尾草綠、米杏與淺暖灰。
- 各章節可保留自身場景特色，但不偏離淡藍天空與低飽和柔和氛圍。

## 平台與支援

- Web：Desktop / iPad / iPhone / Android
- PWA：可安裝

## 技術總覽

React + TypeScript + Vite / Phaser 3 / GSAP / Zustand / React Router / Firebase / Vercel（**無音訊**）。
